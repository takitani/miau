# UX-10: Multi-Language (i18n)

## Overview

Support multiple languages for the user interface.

## Technical Requirements

```go
package i18n

var translations = map[string]map[string]string{
    "en": {
        "inbox": "Inbox",
        "compose": "Compose",
        "reply": "Reply",
        "archive": "Archive",
    },
    "pt-BR": {
        "inbox": "Caixa de Entrada",
        "compose": "Nova Mensagem",
        "reply": "Responder",
        "archive": "Arquivar",
    },
    "es": {
        "inbox": "Bandeja de entrada",
        "compose": "Redactar",
        "reply": "Responder",
        "archive": "Archivar",
    },
}

func T(key string) string {
    lang := GetCurrentLanguage()
    if trans, ok := translations[lang][key]; ok {
        return trans
    }
    return translations["en"][key]
}
```

## Acceptance Criteria

- [ ] English (default)
- [ ] Portuguese (pt-BR)
- [ ] Spanish
- [ ] Language persisted in config

## Estimated Complexity

Low-Medium
